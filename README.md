# carocledc3/csc-campaign

A simple website for the campaign of Carlos Romeo C. Del Castillo III for CITCS-CSC Councillor. Built with [TailwindCSS](https://tailwindcss.com/) and [DaisyUI](https://daisyui.com/).

Carlos Romeo is endorsed by the Sinag Partylist:
- **Gemwil Salayog** for **Governor**
- **Eunice Cueno** for **Vice Governor**
- **Nestor Abenoja** for **Ombudsman**
- **Jade Rillera** & **Carlos Romeo Del Castillo** for **Councillors**

Carlos Romeo also endorses the following candidates to the University Student Council:
- **Sherline Balla** for **USC President**
- **Cherry Doyaoen** for **USC Senator, CITCS**
